# Richfield Connect

## Overview

Richfield Connect is a student-focused social networking platform developed to enhance communication, collaboration, and digital engagement among Richfield students. The platform provides a centralized environment where students can create profiles, share posts, interact with content, and build connections within the Richfield community.

This project was developed using HTML, CSS, JavaScript, jQuery, and Local Storage to simulate the core functionality of a social media platform.

---

## Project Objectives

The aim of Richfield Connect is to:

* Promote student engagement and networking.
* Provide a platform for students to share ideas and opinions.
* Allow students to create personalized profiles.
* Demonstrate front-end web development concepts and user interaction techniques.
* Apply HTML, CSS, JavaScript, jQuery, and browser storage technologies in a practical project.

---

## Features

### Home Page

* Welcomes users to the platform.
* Introduces the purpose of Richfield Connect.
* Highlights key platform features.
* Provides quick access to the Sign-Up page.

### About Us Page

* Explains the purpose of Richfield Connect.
* Describes how the platform encourages student communication and community building.

### User Registration

Users can:

* Register with:

  * Email address
  * Full name
  * Password
  * Student number
  * Campus selection
  * Interests
  * Personal biography

Additional functionality:

* Password confirmation validation.
* Required field validation.
* Error messages for incomplete information.

### Live Profile Preview

A unique feature that displays user information in real time as they complete the registration form.

The live preview updates automatically using jQuery event listeners, allowing users to see how their profile will appear before saving.

### Profile Page

Displays saved user information including:

* Email
* Name
* Student Number
* Campus
* Interests
* Biography

The page also handles situations where no profile has been created.

### Student Feed

Users can:

* Create posts.
* View their posted content.
* Like posts using an interactive like counter.
* Delete posts from the feed.

Additional enhancements:

* Like button animation.
* Dynamic content updates.
* Input validation to prevent empty posts.
